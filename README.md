#  UnAd-IOS Ultra
### A minimal smooth working Ad-Blocker & Zen Reading Engine created by Adil.

UnAd-IOS Ultra is a high-performance Chrome extension designed to neutralize digital distractions and provide a seamless, and optimized browsing experience. Featuring a modern glassmorphism UI and a powerful content extraction engine.

---

## Key Features

* **Neural Protection:** A centralized toggle to enable or disable global ad-blocking rules instantly.
* **Zen Mode (BETA):** A distraction-free reading engine that extracts core article content and applies high-end typography (Georgia Serif) for a premium reading experience.
* **Futuristic-Themed Interface:** Modern, rounded UI with a pulsing "Core" indicator showing real-time system status.
* **Precision Blocking:** Context-menu integration allowing users to manually hide any remaining annoying elements with a right-click.
* **Domain Whitelisting:** A dedicated dashboard to manage sites where you wish to support creators by allowing ads.

---

## Installation Guide

Since this is a Pro developer build, you must load it as an "Unpacked" extension:

1.  **Download the Repository:** Clone this repo or download the ZIP and extract it to a folder named `UnAd-IOS`.
2.  **Open Chrome Extensions:** Navigate to `chrome://extensions` in your browser.
3.  **Enable Developer Mode:** Toggle the switch in the top-right corner.
4.  **Load Unpacked:** Click the **"Load unpacked"** button and select the `UnAd-IOS` folder.
5.  **Pin for Easy Access:** Click the puzzle icon in your toolbar and pin UnAd-IOS Ultra for quick access to the Ad-Blocker.

## How to Use

### 1. The Ad-Blocker (Popup)
Click the extension icon to open the Hub. 
* **Protection Toggle:** Use the switch at the top to pause or resume all blocking.
* **Zen Mode (BETA):** Click "Enter Zen Mode" while on a news article or blog to strip away sidebars, ads, and clutter.
* **Counter:** The central display shows the total number of intrusive elements neutralized during your current session.

### 2. Manual Neutralization
See an element the Ad-Blocker missed? 
* **Right-click** on the element.
* Select **"UnAd-IOS: Block Element"** to remove it instantly.

### 3. Managing the Whitelist
If a site doesn't load correctly or you want to see ads:
* Open the Popup and click **Settings**.
* Add the domain (e.g., `youtube.com`) to your whitelist.

## ⚠️ Important Notes

* **System Pages:** Chrome security prevents extensions from running on `chrome://` pages or the Chrome Web Store.
* **Refresh Required:** After first installation, please refresh your open tabs to allow the Ad-Blocker to initialize.
* **Zen Mode (BETA):** **Zen mode** is **BETA** meaning it can have errors.
